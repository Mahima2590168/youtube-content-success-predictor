# 🎬 YouTube Content Success Predictor

A Machine Learning project that predicts whether a YouTube video will perform well based on its title, duration, and metadata.

---

## 🚀 Features
- NLP-based sentiment analysis on video titles
- TF-IDF keyword extraction
- Multiple ML models (Logistic, Random Forest, XGBoost)
- FastAPI backend for predictions
- Streamlit interactive dashboard

---

## 🧠 Tech Stack
- Python
- Scikit-learn
- XGBoost
- FastAPI
- Streamlit
- NLP (TextBlob)

---

## 📊 Dataset
Custom YouTube dataset with:
- title
- views
- duration
- tags

---
